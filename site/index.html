Native BYOND on Fedora

Install BYOND's native Linux compiler and hosting tools via Fedora's package manager!

Installing the repository via DNF

Installation via DNF can be done either graphically by clicking on the relevant link for your version of Fedora:

  • Fedora 32 - Beta
  • Fedora 31
  • Fedora 30

Alternatively, you can install via the command line with:

sudo dnf install -y "https://fedora.byondlabs.io/$(. /etc/os-release; echo ${VERSION_ID})/base/x86_64/Packages/byondlabs-release-$(. /etc/os-release; echo ${VERSION_ID})-1.noarch.rpm"

Packages

This repository currently provides two packages:

  • byond-dreammaker for the Linux DM compiler

  • byond-dreamdaemon for the Linux BYOND world host

In addition, there is an install group that can be used to get all BYOND packages:

sudo dnf groupinstall BYOND
